# Slice 2.5 — Pulido de UX de lectura · Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Goal:** Mejorar la ergonomía de lectura siguiendo patrones de Royal Road/AO3/Medium: navegación de capítulo arriba y abajo, salto directo entre capítulos, barra de progreso, tiempo de lectura estimado y enlace de accesibilidad "saltar al contenido".

**Architecture:** Un componente `ChapterNav.astro` reutilizable (prev/next + `<select>` nativo de salto) se coloca al inicio y al final de la vista de lectura. La barra de progreso es CSS + un script inline mínimo. El tiempo de lectura es una función pura (`readingTime`) calculada en build y probada con Vitest. Cero JS de framework.

**Tech Stack:** Astro, `<select>` nativo, CSS, Vitest.

## Global Constraints
- Cero JS de framework; controles nativos.
- Accesibilidad: foco visible, `aria-label`, skip-link, `prefers-reduced-motion`.
- Correr `npm run sync` (o usar contenido en disco) antes de build.

  });
  it('nunca baja de 1 minuto', () => {
    expect(readingTime('')).toBe(1);
    expect(readingTime('hola mundo')).toBe(1);

- [ ] **Step 2: Ver fallar** — `npx vitest run src/lib/reading.test.ts` → FAIL.

- [ ] **Step 3: Implementar `web/src/lib/reading.ts`**
```ts
export function readingTime(text: string, wpm = 200): number {
  const words = text.trim().split(/\s+/).filter(Boolean).length;
  return Math.max(1, Math.ceil(words / wpm));
}
```

- [ ] **Step 4: Ver pasar** — `npx vitest run src/lib/reading.test.ts` → PASS.

### Task 2: Componente `ChapterNav`

**Files:** Create `web/src/components/ChapterNav.astro`

**Interfaces:** Props `{ novelSlug: string; current: string; items: { chapterSlug: string; title: string }[] }`. Renderiza prev/next + `<select>` que navega al cambiar.

- [ ] **Step 1: Crear `web/src/components/ChapterNav.astro`**
```astro
---
interface Item { chapterSlug: string; title: string; }
interface Props { novelSlug: string; current: string; items: Item[]; }
const { novelSlug, current, items } = Astro.props;
const idx = items.findIndex((i) => i.chapterSlug === current);
const prev = items[idx - 1];
const next = items[idx + 1];
const base = `/novela/${novelSlug}`;
---
<nav class="chapter-nav" aria-label="Navegación de capítulos">
  {prev
    ? <a class="cn-btn" href={`${base}/${prev.chapterSlug}`} rel="prev">← Anterior</a>
    : <span class="cn-btn cn-disabled" aria-disabled="true">← Anterior</span>}

  <select class="cn-select" aria-label="Saltar a capítulo" data-base={base}>
    {items.map((i) => (
      <option value={i.chapterSlug} selected={i.chapterSlug === current}>{i.title}</option>
    ))}
  </select>

  {next
    ? <a class="cn-btn" href={`${base}/${next.chapterSlug}`} rel="next">Siguiente →</a>
    : <span class="cn-btn cn-disabled" aria-disabled="true">Siguiente →</span>}
</nav>

<style>
  .chapter-nav {
    display: flex; align-items: center; gap: var(--space-3);
    font-family: var(--font-sans);
  }
  .cn-btn {
    padding: 0.4rem 0.7rem; border: 1px solid var(--border);
    border-radius: var(--radius-sm); white-space: nowrap; color: var(--accent);
  }
  .cn-btn:hover { background: var(--surface-2); }
  .cn-disabled { color: var(--text-muted); opacity: 0.5; }
  .cn-select {
    flex: 1; min-width: 0; padding: 0.4rem 0.6rem;
    background: var(--surface); color: var(--text);
    border: 1px solid var(--border); border-radius: var(--radius-sm);
    font-family: var(--font-sans); font-size: var(--step--1);
  }
</style>

<script>
  document.querySelectorAll('.cn-select').forEach((el) => {
    const sel = el as HTMLSelectElement;
    sel.addEventListener('change', () => {
      const base = sel.dataset.base;
      if (base) location.href = `${base}/${sel.value}`;
    });
  });
</script>
```

- [ ] **Step 2: Reescribir `web/src/pages/novela/[slug]/[capitulo].astro`**
```astro
---
import { getCollection, type CollectionEntry } from 'astro:content';
import BaseLayout from '../../../layouts/BaseLayout.astro';
import ChapterNav from '../../../components/ChapterNav.astro';
import { readingTime } from '../../../lib/reading';

export async function getStaticPaths() {
  const chapters = await getCollection('chapters');
  return chapters.map((c) => ({
    params: { slug: c.data.novelSlug, capitulo: c.data.chapterSlug },
    props: { entry: c },
  }));
}

interface Props { entry: CollectionEntry<'chapters'>; }
const { entry } = Astro.props;
const { slug } = Astro.params;
const { Content } = await entry.render();

const items = (await getCollection('chapters'))
  .filter((c) => c.data.novelSlug === slug)
  .sort((a, b) => a.data.number - b.data.number)
  .map((c) => ({ chapterSlug: c.data.chapterSlug, title: c.data.title }));

const minutes = readingTime(entry.body);
---

<BaseLayout title={`${entry.data.title} — Draveir`}>
  <div class="reading-progress" id="reading-progress"></div>
  <main class="container reading" id="contenido">
    <div class="reading-top">
      <a href={`/novela/${slug}`}>← Índice</a>
      <ChapterNav novelSlug={slug} current={entry.data.chapterSlug} items={items} />
    </div>
    <h1>{entry.data.title}</h1>
    <p class="reading-meta">~{minutes} min de lectura</p>
    <article><Content /></article>
    <ChapterNav novelSlug={slug} current={entry.data.chapterSlug} items={items} />
  </main>
  <script is:inline>
    (function () {
      var bar = document.getElementById('reading-progress');
      var article = document.querySelector('.reading article');
      if (!bar || !article) return;
      function update() {
        var rect = article.getBoundingClientRect();
        var total = rect.height - window.innerHeight;
        var scrolled = Math.min(Math.max(-rect.top, 0), total);
        bar.style.width = (total > 0 ? (scrolled / total) * 100 : 0) + '%';
      }
      document.addEventListener('scroll', update, { passive: true });
      window.addEventListener('resize', update);
      update();
    })();
  </script>
</BaseLayout>
```
